CSS HEAD INSPECTOR

Click the button above to toggle the inspector styles.

What is this?

This is an small set of css rules revealing hidden html elements mostly inside head. Applying this styles to any existing website you can show most of html's head contents such as <title>, <meta>, <link> or <style>

Revealed elements are bright colored in order to stand out in most of website designs.

Bookmarklet

You can easily use the inspector on every site using a bookmarklet link.

The bookmarlet comes in two flavors, one injects the stylesheet using a <link> and the other using <style> inlining all the css code, the <style> version is useful to prevent CSP crossdomain errors.

Motivation

This project is just a tiny toy, it started as an experiment in codepen. I decided to create this bookmarlet so maybe someone might find it useful for some quick checks about what is on the head of a website. But... we already have dev tools... Yep! It does not pretend to be as useful as a full featured dev tools, it is what it is and you can use it as you like :)
