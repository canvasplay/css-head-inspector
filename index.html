CSS HEAD INSPECTOR

Pluggable CSS styles to visually inspect hidden html's head elements.

MIT LICENSE v0.3.2

What is this? Injctable CSS styles reavealing html's head contents

This is an small set of css rules revealing typically hidden elements mostly inside html's head. This is where the plugin's name comes from. The css rules targets any of the following html elements:

  • <html>
  • <head>
  • <body>
  • <title>
  • <meta>
  • <link>
  • <script>
  • <style>

Once injected, these elements will be visible and will also show relevant information about theme such as a link's href or an script's source file. Revealed elements are bright colored and super high z-indexed in order to stand out in most of website designs.

Usage How to use it?

This is meant to be used as an injectable CSS plugin. In order to provide an easy way of plug and unplug the styles on any website you can use one of the following approaches:

Bookmarklet

By using the bookmarlet it will create and inject a new <link> in the current page linking to the latest styles version. By clicking the bookmarlet twice it will remove any previously injected styles, this way you can use the bookmarlet as an inspect toogle button.

To "install" the bookmarlet you can drag and drop the link below in you browser's bookmarks toolbar:

CSS HEAD INSPECTOR

The Google Chrome™ extension

There is also a Google Chrome™ extension, it does exactly the same as the bookmarlet but in a more elegant and integrated way. Unfortunately the extension is not published in the chrome store, so if you want to use it you have to follow these steps:

  1. Download or clone the repository from github.
  2. Navigate with your chrome browser to chrome://extensions/
  3. Activate the developer mode
  4. Press the Load unpacked extension button
  5. Select the chrome-extension folder inside the downloaded package

And violá! You got now an easy to use button next to your browser url bar. Click for toogling the inspector styles on any website you visit.
